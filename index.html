Encontrar o número de vezes que uma substring aparece como subsequência de uma string

Entrada:

uma string mainstring representando a string principal na qual as subsequencias vão ser procuradas

uma string substring representando a string que vai ser procurada como subsequencia da string principal

um inteiro m representando o tamanho da mainstring

um inteiro n representando o tamanho da substring

Saida:

Um inteiro representando a quantidade de vezes que a substring aparece como subsequencia da mainstring

O objetivo do problema é encontrar o número de vezes que uma substring aparece como subsequência de uma string principal.

Para uma string principal “eu programo e eu amo isso” e uma substring a ser procurada “eu”, a saída esperada é 4 pois as letras da substring “eu” aparecem em sequência 4 vezes na string principal.

eu programo e eu amo isso

eu programo e eu amo isso

eu programo e eu amo isso

eu programo e eu amo isso

As letras em negrito acima podem ajudar a visualizar as sequências das letras de “eu”.

É importante notar que o algoritmo é case-sensitive portanto se a substring fosse “Eu” haveria 0 ocorrências de subsequências na string principal.
